What's APN Assistant

This app helps you about mobile network settings when you offline.

Sometimes, you choose SIM card of minor operator that called MVNO.

And you have to set "APN", via downloading configuration profiles from Safari, receive E-Mail.

It's need Internet, almost case using free Wi-Fi network.

But, you can not all find free Wi-Fi while traveling.

This app can set new APN even you're devices can't access to Internet.

So, if you get this app, your travel becomes better than now!!
